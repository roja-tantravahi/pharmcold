# 🌡️ PharmaCold — Real-Time Cold Chain Integrity Platform

A production-grade **PySpark + Azure** data engineering project that predicts pharmaceutical spoilage before it happens, enforces regulatory audit trails, and exposes hidden risk nodes in a global logistics network.

---

## Problem Statement

Pharmaceutical companies lose **$35 billion annually** to cold chain failures. Vaccines, biologics, and temperature-sensitive drugs spoil in transit due to invisible temperature excursions.

This platform predicts spoilage probability **2–6 hours in advance** using real-time IoT sensor streams, auto-generates FDA-compliant audit trails, and identifies high-risk logistics nodes using graph analytics.
